# Standard Review

## Instructions

1. 先确认输入文档已经解析为 Markdown 或结构化文本。
2. 审核至少覆盖：文档结构、范围描述、术语定义、引用文件、条款一致性、可追溯依据。
3. 每条发现必须包含严重级别、位置、问题、建议和证据。
4. 不能从模型常识伪造标准条款；无法定位证据时标记“依据不足”。
5. 最终报告写入 `/workspace/output/`，文件名应包含任务类型和时间戳。
